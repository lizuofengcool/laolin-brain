# 个人私有第二大脑

一款基于 Next.js 16 构建的全功能智能文件管理与知识库系统，支持本地存储和云端部署双模式。它不仅仅是一个网盘，而是一个能"理解"你文件内容的 AI 驱动知识管理平台。

## ✨ 核心特性

### 🏢 SaaS 多租户架构
- **完整多租户隔离**：基于 tenantId 的底层数据隔离，所有业务表强制携带租户标识
- **租户管理后台**：运营后台支持租户列表、详情查看、状态管理、套餐调整
- **套餐订阅系统**：免费版/专业版/企业版多档位，支持订阅升级、降级、取消
- **配额控制**：存储容量配额、AI调用次数配额，自动统计与超限提醒
- **订单管理**：完整的订单生命周期管理，支持支付宝、微信支付
- **试用机制**：新用户免费试用，试用期到期自动降级

### 📁 多格式文件管理
- 支持图片、文档（Word/PDF/PPT）、Markdown、文本等全类型文件
- 拖拽排序、文件夹树管理、标签系统
- 文件版本控制，支持历史版本回溯与恢复
- 软删除回收站，防止误删

### 🤖 AI 智能助理
- **自动摘要**：智能提取文档核心内容
- **OCR 识别**：图片文字提取
- **图像理解**：AI 图片描述与问答
- **语义搜索**：基于向量的相似度搜索
- **智能关联推荐**：自动发现相关文档
- **知识图谱**：文件关系可视化
- **AI 使用量统计**：每日调用限额与使用追踪

### 🔒 隐私安全
- 本地 IndexedDB 存储 + 云端加密双模式
- 密码保护和分享链接
- HMAC-SHA256 签名令牌认证
- 端到端加密传输
- 多租户数据物理隔离

### 🖥️ 全平台适配
- PWA 离线支持
- 手势操作（滑动/长按/缩放）
- 响应式设计，移动端完美适配
- **Tauri 桌面端**：原生性能，本地 SQLite 存储，支持多租户

### 📊 数据洞察
- 存储分析仪表板（Recharts）
- 知识图谱可视化（SVG 力导向图）
- 时间线浏览
- 文件类型分布、增长趋势预测

### ⚡ 高级功能
- 文件版本管理
- 自动化规则引擎
- 批量操作
- 数据备份恢复（支持校验和验证）
- 键盘快捷键
- 人脸分组与识别

### ☁️ 云同步引擎
- **增量同步**：只同步变更文件，节省带宽
- **冲突检测与处理**：自动检测版本冲突，支持手动/自动解决
- **离线队列**：离线操作自动排队，联网后自动同步
- **多租户隔离**：每个租户独立同步状态与队列
- 阿里云OSS + Cloudflare R2 双存储支持

### 💳 支付系统
- 支付宝、微信支付接入框架
- 支持订阅升级、订单管理
- 支付签名验证，保障交易安全
- 幂等性设计，防止重复支付

### 架构概览

```
┌─────────────────────────────────────────────────────────┐
│                        前端层                           │
│  Next.js 16 + React 19 + Tailwind CSS 4 + shadcn/ui    │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                        API层                            │
│  Next.js API Routes + 认证中间件 + 多租户上下文         │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                      数据访问层                         │
│  TenantDb 封装 ── 强制 tenantId 过滤 ── Prisma ORM     │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│                        存储层                           │
│  SQLite (云端) / IndexedDB (浏览器) / SQLite (Tauri)   │
└─────────────────────────────────────────────────────────┘
```

### 多租户架构设计

- **数据隔离**：所有业务表携带 `tenantId` 字段，底层强制过滤
- **租户上下文**：`getTenantDbFromRequest()` 自动获取租户DB实例
- **租户-用户关系**：支持多用户属于同一租户，租户成员管理
- **配额管理**：租户级别的存储、AI 调用配额控制
- **订阅系统**：租户级别的订阅状态与套餐管理

## 🚀 快速开始

### 环境要求
- Node.js >= 20.0.0
- Rust >= 1.70（桌面端开发）
- 包管理器：pnpm / bun / npm

### 安装

```bash
git clone <你的仓库地址>
cd laolin-brain
pnpm install
```

### 本地运行

```bash
pnpm dev
# 访问 http://localhost:3000
```

### 生产构建

```bash
pnpm build
pnpm start
```

### Tauri 桌面端开发

```bash
# 开发模式
pnpm tauri dev

# 构建生产版本
pnpm tauri build
```

## 🧪 测试

```bash
# 运行单元测试
pnpm test

# 监听模式
pnpm test:watch

# 运行类型检查
pnpm type-check
```

## 📋 功能概览

### 15+ 个视图

1. **仪表板** - 文件统计、存储图表、最近文件
2. **文件管理** - 上传/浏览/搜索/批量操作
3. **相册** - 网格/瀑布流布局、图片灯箱、人脸识别
4. **时间线** - 按时间线浏览文件
5. **收藏夹** - 收藏文件分组展示
6. **标签** - 标签管理和筛选
7. **搜索** - 全文搜索 + AI 问答 + 语义搜索
8. **回收站** - 文件恢复/永久删除
9. **存储分析** - Recharts 图表可视化
10. **知识图谱** - SVG 力导向关联图
11. **设置** - 账户/存储/备份/自动化/云同步/会员
12. **会员中心** - 订阅状态、套餐对比、订单历史
13. **运营后台** - 租户管理、订单管理、系统设置
14. **云同步** - 增量同步、冲突解决、离线队列
15. **人脸管理** - 人脸分组、照片管理

- AI：摘要、OCR、图像理解、问答、关联推荐、知识图谱、使用量
- 支付：创建支付、回调、状态查询
- 账单：订阅、订单、套餐
- 云同步：同步、状态、冲突、队列、备份、配置

- 管理后台：仪表盘、租户、订单、设置
- 其他：搜索、语义搜索、存储分析、备份
